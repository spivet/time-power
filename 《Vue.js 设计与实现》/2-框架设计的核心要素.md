## 提升用户开发体验

### 友好的警告提示

在使用库或者框架的时候，提供友好的警告信息至关重要，这可以帮助开发者快速定位问题。

```javascript
// 定义一个简单的 warn 函数
function warn(msg) {
    console.warn(`[xxx warn] ${msg}`)
}

class Poster {
    constructor() {}
    muntCanvas(canvas, el) {
        const el = document.querySelector(selector)
        if (!el) {
            warn(`${el} 元素不存在`)
        }
        if (canvas.tagName.toLowerCase() !== 'canvas') {
            warn(`${el} `)
        }
        (el || document.body).appendChild(canvas)
    }
}
```
